How to shape your SurrealDB query

Return a single array that mixes node rows and edge rows.

Node row: { id, label } (or title/text as label fallback)

Edge row: { id, in, out, label }

RETURN array::concat(
    -- Nodes
    (SELECT id, title AS label FROM your_node_table),
    (SELECT id, text  AS label FROM another_node_table),

    -- Edges / relations
    (SELECT id, in, out, 'relation_name' AS label FROM your_relation_table)
);

Tip: each node id should be unique (e.g. document:1), and edge in/out should point to existing node ids.
